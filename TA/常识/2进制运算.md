XOR 的基本概念

**XOR** 全称是 **Exclusive OR（异或）**。

逻辑规则是：当且仅当两个输入 **不同** 时，结果为 1；相同为 0。

Morton函数：

    //本质：32 位整数中，提取出所有偶数位，并压缩到连续的低 16 位。  
    uint Compact1By1(uint x)  
    {  
       x &= 0x55555555;                  // 0x55555555  =>0101010101010101010 为了保留偶数位的值，所以用555作为掩码  
	   x = (x ^ (x >>  1)) & 0x33333333; // XOR操作，3333=>00110011001100110011 ,因为之前的数据已经满足 0a0b0c0d 那么接下来通过运算得到 00ab 00cd以此类推，将ab cd保留在后两位  
       x = (x ^ (x >>  2)) & 0x0f0f0f0f; // 0x0f0f0f0f  =>0000111100001111     
     上述结果为 00ab 00cd ^ 0000 ab00... => 00ab abcd... 再参与&运算 => 0000 abcd  
        x = (x ^ (x >>  4)) & 0x00ff00ff; // 0000 0000 1111 1111 => 0000 abcd 0000 efgh >>4 = 0000 0000 abcd 0000 efgh ^ 0000 abcd 0000 efgh   
     = 0000 abcd abcd efgh efgh => 0000 0000 abcd efgh  
     x = (x ^ (x >>  8)) & 0x0000ffff; // 以此类推 ,这里变为16位  
      return x;  
    }
