## 启用内存

**

    sudo fallocate -l 2G /swapfile

设置正确权限

    sudo chmod 600 /swapfile

格式化为 swap 类型：

    sudo mkswap /swapfile

启用 swap 文件：

    sudo swapon /swapfile


确认是否启用成功：

    swapon --show
    free -h


让 swap 文件开机自动启用

编辑 /etc/fstab：

    sudo nano /etc/fstab


添加这一行到文件最后：

    /swapfile none swap sw 0 0
